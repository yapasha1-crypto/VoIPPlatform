API Reference

RESTful API with comprehensive endpoints for VoIP, SMS, and account management. All requests are authenticated via API keys and return JSON responses.

Example: Send SMS

POST /api/v1/sms/send 
{
  "to": "+14155551234",
  "from": "+14155559999",
  "message": "Hello from Beroea!",
  "callback_url": "https://your-app.com/webhook"
}

Response (200 OK)

{
  "message_id": "msg_abc123",
  "status": "queued",
  "to": "+14155551234",
  "segments": 1,
  "cost": 0.0075
}

Core Endpoints

  • POST /api/v1/calls/initiate
  • POST /api/v1/sms/send
  • GET /api/v1/numbers/available
  • GET /api/v1/rates/lookup
  • DELETE /api/v1/numbers/{id}

Features

  • RESTful JSON API
  • OAuth 2.0 & API Keys
  • Webhooks for Events
  • Rate Limiting: 1000 req/min
